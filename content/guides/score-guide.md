## 청약 가점제란?

청약 가점제는 주택청약 시 무주택기간, 부양가족 수, 청약통장 가입기간에 따라 점수를 부여하여 당첨자를 결정하는 방식입니다. 2000년 도입 이후 현재까지 민영주택 일반공급의 75~100%에 적용되고 있습니다.

가점제의 핵심은 **무주택 기간이 길고, 부양가족이 많고, 청약통장 가입기간이 길수록 유리**하다는 점입니다.

## 가점 항목과 배점

가점 총점은 **84점 만점**이며, 다음 3가지 항목으로 구성됩니다.

| 항목 | 최대 점수 | 기준 |
|---|---|---|
| 무주택 기간 | 32점 | 1년 미만 2점, 15년 이상 32점 |
| 부양가족 수 | 35점 | 0명 5점, 6명 이상 35점 |
| 청약통장 가입기간 | 17점 | 6개월 미만 1점, 15년 이상 17점 |

## 1. 무주택 기간 점수 (최대 32점)

무주택 기간은 세대주 기준으로 계산합니다. **기혼자는 혼인신고일부터, 미혼자는 만 30세 생일부터** 무주택 기간을 인정합니다.

### 점수표

| 무주택 기간 | 점수 |
|---|---|
| 1년 미만 | 2점 |
| 1년~2년 | 4점 |
| 2년~3년 | 6점 |
| 3년~4년 | 8점 |
| 4년~5년 | 10점 |
| 5년~6년 | 12점 |
| 6년~7년 | 14점 |
| 7년~8년 | 16점 |
| 8년~9년 | 18점 |
| 9년~10년 | 20점 |
| 10년~11년 | 22점 |
| 11년~12년 | 24점 |
| 12년~13년 | 26점 |
| 13년~14년 | 28점 |
| 14년~15년 | 30점 |
| 15년 이상 | 32점 |

### 주의사항

- 세대원 중 한 명이라도 주택을 소유하면 무주택기간이 0으로 초기화됩니다
- 분양권, 입주권도 주택으로 간주합니다
- 상속으로 취득한 주택은 3개월 내 처분 시 무주택으로 인정될 수 있습니다

## 2. 부양가족 수 점수 (최대 35점)

부양가족은 세대주 본인을 제외한 세대원 중 무주택인 직계존비속입니다.

| 부양가족 수 | 점수 |
|---|---|
| 0명 | 5점 |
| 1명 | 10점 |
| 2명 | 15점 |
| 3명 | 20점 |
| 4명 | 25점 |
| 5명 | 30점 |
| 6명 이상 | 35점 |

### 부양가족으로 인정되는 경우

- **배우자**: 무주택 여부 무관하게 1명으로 인정
- **직계비속(자녀, 손자녀)**: 무주택인 경우 인정
- **직계존속(부모, 조부모)**: 무주택이며 3년 이상 동일 주민등록 세대원인 경우 인정

### 부양가족으로 인정되지 않는 경우

- 형제, 자매는 제외
- 주택을 소유한 직계존비속
- 별도 세대를 구성한 자녀

## 3. 청약통장 가입기간 점수 (최대 17점)

청약통장의 종류와 무관하게 가입일로부터 경과된 기간으로 점수를 계산합니다.

| 가입기간 | 점수 |
|---|---|
| 6개월 미만 | 1점 |
| 6개월~1년 | 2점 |
| 1년~2년 | 3점 |
| 2년~3년 | 4점 |
| 3년~15년 | 5~16점 (1년마다 1점 증가) |
| 15년 이상 | 17점 |

## 가점 전략

### 단기 전략 (1~3년)
부양가족 가점이 가장 빠르게 올릴 수 있는 항목입니다. 결혼을 통한 배우자 추가, 자녀 출산으로 부양가족을 늘릴 수 있습니다.

### 중기 전략 (3~10년)
무주택 기간을 유지하면서 점수를 꾸준히 쌓는 전략입니다. 주택 구매 충동을 억제하고 전세·월세로 거주하는 방식입니다.

### 장기 전략 (10년 이상)
무주택기간 32점 만점(15년)과 통장기간 17점 만점(15년)을 목표로 꾸준히 가점을 쌓는 방법입니다. 부양가족이 많을수록 만점에 가까워집니다.

> **현실적인 목표**: 수도권 일반 단지 평균 커트라인이 50~60점 수준입니다. 무주택기간 16점(7~8년) + 부양가족 3명 20점 + 통장기간 8점(6~7년) = 44점 정도라면 경기도 비인기 지역부터 도전해볼 수 있습니다.
